# 📘 DOVIAI 课程体系方案

（采用 LRMI + Schema.org + AEO 最佳实践）

## 1. 课程层级（educationalLevel）

| Level                  | 描述        | 商业方式        |
| ---------------------- | --------- | ----------- |
| **Beginner（免费入门）**     | 软件基础、快速理解 | 全部免费        |
| **Intermediate（进阶实战）** | 技能训练、任务练习 | 单买 或 会员可看   |
| **Advanced（高阶训练）**     | 企业项目、商业案例 | 单买 + 会员 9 折 |

---

## 2. LRMI 教学类型（educationalUse）

### 🟦 Beginner — 入门体验课（免费）

适合：吸引新用户、建立信任
LRMI：

- **Lecture**（讲授类）
- **Demonstration**（演示）
- **Exercise**（练习）
- **Introduction**（课程导入类）

Schema 作用：告诉搜索引擎“这是入门 + 免费 + 教师讲解类内容”，更容易匹配“教程/讲解”类搜索。

---

### 🟩 Intermediate — 进阶实战课（可单买/会员可看）

适合：技能提升、可量化训练
LRMI：

- **Lesson**（课时单元）
- **Practice**（技能练习）
- **Exercise**（实操任务）

---

### 🟥 Advanced — 高阶训练课（商业项目、付费）

适合：职业准备、就业转型
LRMI：

- **Project**（综合项目）
- **CaseStudy**（商业案例）
- **Simulation**（模拟真实工作场景）

Schema 作用：强信号告诉搜索引擎这是“职业训练”类内容，有利于 AEO 职业路径推荐。

---

## 3. Program（体系课）

### 🧩 Program A：会员实战训练体系（由 Intermediate 组成）

- 可由多个进阶实战课组合
- 会员可完整学习
  LRMI：
- **Curriculum**（课程路径）
- **Unit**（子模块）

---

### 🧩 Program B：职业技能训练体系（Advanced 部分）

- 面向想提升职业竞争力的用户
- 单买，不包含在会员内（增强变现能力）
  LRMI：
- **ProfessionalDevelopment**
- **Curriculum**

---
